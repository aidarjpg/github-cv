# GitHub Pages CV

This folder is ready to publish as a GitHub Pages site.

## Files
- `index.html`
- `style.css`

## Publish
1. Push this folder to a GitHub repository.
2. Open repository `Settings` -> `Pages`.
3. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` and `/ (root)`
4. Save and wait 1-2 minutes.
5. Open the generated URL shown in the Pages settings.

## Important
- Replace placeholder `LinkedIn` and `GitHub` links in `index.html`.
- If you want PDF download button or project/publication links, add them to `index.html`.
